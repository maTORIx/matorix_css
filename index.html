matorix css

Custom CSS Design Kit

概要

matorix cssは @matorixmatrixによって作成された、カスタムのCSSセットです。白と黒をベースとしたシンプルなデザインを目指して作成しています。

著作権はcc0です。ご自由に利用ください。

なお、このCSSはmatorixmatrixによる個人利用を主な目的として作成しています。サポートや説明などは特にないため、ご注意ください。

このキットではCSSのベース部分を統一し、綺麗にすることを最大の目的としています。このCSSを読み込むだけでもWebサイトはある程度綺麗になるようにしています。


利用方法

下記のhtmlをヘッダーに埋め込んでください

<link rel="stylesheet" href="https://matorix.github.io/matorix_css/style.css">
